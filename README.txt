How to run:
To run the game first run the server, then run a client enter name and select a colour.
you will see a "Waiting for player" message , launch a second client and enter a second name and colour.

to play the game enter a number between 1 and 9 to select a column. When the winnning streak is met each player receives a message depending on the outcome and disconnects.

Unit Tests:
Unit tests where created using Mockito and Junit in sclipse , current coverage is 94%
